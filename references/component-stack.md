# Stack (VStack, HStack, ZStack)

SkipUI supports SwiftUI stacks on Android. Use `VStack`, `HStack`, `ZStack` with optional spacing/alignment. Alignment options (e.g. `.leading`, `.center`, `.trailing` for HStack) are supported. Stacks map to Compose `Column`, `Row`, and `Box`. Use `Spacer()` for flexible space. Layout behavior follows SwiftUI semantics; combine with `frame`, `padding`, and other modifiers as in SwiftUI.
