# Button (SwiftUI)

SkipUI supports SwiftUI `Button` on Android. Use the standard initializers: `Button(action:label:)`, `Button(_:action:)`. Supported roles (e.g. `.destructive`) and styles: `.plain`, `.bordered`, `.borderedProminent`. Use `.disabled(_:)` to disable. Buttons with images should include text labels for accessibility. Behavior and appearance match SwiftUI semantics; on Android they map to Compose Material components.
